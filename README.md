# 🧴 AI Skin Analysis

An AI-powered skin analysis web application that analyzes facial images and provides insights about acne, redness, skin health, and personalized skincare recommendations.

  Features
- 📸 Upload facial images
- 🤖 AI-based skin analysis
- 🔍 Acne detection
- 🌡️ Redness detection
- 📊 Skin health score calculation
- ⚠️ Severity level prediction
- 💡 Personalized skincare recommendations
- 📄 Downloadable PDF reports
- 📱 Responsive and modern web interface
- 🖥️ Built with Flask and OpenCV

 Analysis Dashboard
- Skin Health Score
- Acne Spot Count
- Severity Level
- Personalized Recommendations
- Download Report
